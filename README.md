# 📰 Fake News Detection using BERT

## 📌 Project Description

This project is a web-based application that detects whether a given news text is **Fake** or **Real** using a deep learning model. The system uses a pre-trained transformer model (BERT-based) to analyze the text and provide predictions in real time.

---

## 🎯 Objective

The objective of this project is to:

* Identify fake news using Natural Language Processing (NLP)
* Build a simple and interactive user interface
* Provide real-time predictions

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Transformers (Hugging Face)
* PyTorch

---

## ⚙️ How It Works

1. User enters news text into the application
2. The model processes the input using a transformer-based pipeline
3. The system predicts whether the news is Fake or Real
4. Confidence score is displayed

---

## 🚀 Features

* Easy-to-use interface
* Real-time prediction

1. Install dependencies:

```

```

2. Run the app:

```
python -m streamlit run app.py
```

3. Open the browser link shown in the terminal
